# linux-dev-playbook
Linux setup and configuration via Ansible.

## Installation 
1. [Install Ansible](https://docs.ansible.com/ansible/latest/installation_guide/index.html)
'sudo apt install ansible'  
2. Run ansible  
'ansible-playbook main.yml -i inventory -K'
